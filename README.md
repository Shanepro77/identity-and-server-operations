# Identity and Server Operations

This repository showcases hands-on labs for identity management, Windows server administration, and automation of common administrative tasks. Labs focus on Active Directory, Group Policy, user lifecycle management, and secure access.

## Included Labs

### 01-active-directory-foundations
- Deploy a Windows Server AD domain controller
- Create users and organizational units (OUs)
- Configure basic access control

### 02-gpo-and-access-control
- Implement Group Policy Objects (GPOs)
- Apply security policies to OUs
- Test policy enforcement

### 03-secure-remote-access-powershell
- Enable secure remote access for administrators
- Restrict sessions and permissions
- Audit remote login events

### 04-ad-user-lifecycle-automation
- Automate AD user creation, modification, and removal
- Assign users to groups automatically
- Integrate mailbox creation in Office 365
- Troubleshooting and operational notes included

## Lab Features
- Consistent naming conventions
- Verification of expected behavior
- Documentation of all troubleshooting hurdles
